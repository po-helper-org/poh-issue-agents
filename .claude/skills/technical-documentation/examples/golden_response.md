# Пример: Метод получения карточки элемента
**Endpoint:** GET /api/v1/items/{id}

**Пример:** GET /api/v1/items/42

**Route:** items/get

**Входные параметры (Код vs API):**

| Параметр | Тип | Источник | В коде (Argument) | Описание |
|----------|-----|----------|-------------------|----------|
| id | int | URL Path | $id | ID элемента, извлеченный из {id} компонентом маршрутизации |
| nocache | bool | GET/Query | $nocache | Если 1, обход кэша |

**Логика БД (Deep Dive):**
1. Поиск в `main_db.items_view`: фильтр по `item_id = $id AND is_active = 1`.
2. Если запись не найдена — вернуть 404.
